Clash简介

Clash是一款支持Windows、macOS、Android等多平台的代理软件，支持ss/v2ray/Trojan等，类Surge配置，macOS/Windows/Android上都已经有了图形界面，且召唤师已经支持ss/v2ray/Trojan的Clash订阅，使用起来已经非常简单，十分推荐使用。 简而言之就是一款代理软件，只要有订阅链接就能访问外网！接下来我将介绍如何在Windows，Mac上以及安卓手机上使用Clash。IOS系统暂不支持。

Clash for Windows下载&安装

  1. 源文件在Github, 但基于被墙，小伙伴们可以去Clash For Windows v0.16.3 客户端及汉化补丁 | 六音软件 (sixyin.com)下载 Windows 版本。选择对应的电脑版本就好了。Clash本身不是汉化版本，里面也有对应的汉化教程。

快速上手

  1. 使用方法也很简单，打开 Clash for Windows 应用程序，在左侧的标签页中选择 Profiles ， 在顶部输入您的 Clash 订阅链接，然后点击 Download 按钮。

​

Clash for Windows 会自动拉取配置文件进行更新，如果一切顺利，你应当可以看到成功的提示信息「Success!」，并且下方会自动多出一个文件。

点击新增的配置文件来切换到该配置，然后点击「Proxies」标签页来切换接入点，将顶部的出站模式选择为「Rule」。
此模式下你的网络访问请求将通过 Clash for Winows 进行分流处理。

返回到「General」部分，将「System Proxy」的开关更改为绿色或「On」状态即可开始使用。此外，如果您需要让 Clash for Windows 开机自启，可以将「Start with Windows」也更改为绿色或「On」。

Clash for Mac 下载&安装

  1. 同理Github被墙，网盘安装地址ClashX (lanzoui.com)，选择对应版本。下载安装即可

快速上手

首次使用 ClashX 时，macOS 会提醒你此应用来自未知开发者，请允许打开此应用。
ClashX 首次运行会提示是否安装帮助程序（Helper）。此帮助程序用于设置系统代理，否则每次你通过 ClashX 变更系统系统状态（打开或关闭）时都需要输入密码，请点击 “Install”，之后 macOS 会提示输入用户密码。

复制 Clash 订阅链接。启动 ClashX，点击状态栏中的 ClashX 图标，依次选择「配置」、「托管配置」、「管理」，在弹出的界面点击添加订阅。

点击添加

若干秒后将在软件里看到刚刚添加的配置文件。

点击 ClashX 状态栏图标，将「出站模式」选为「规则判断」，在「Proxy」或「Gloabal」策略组中可以选择自己喜欢的线路，然后点击「设置为系统代理」即可开始使用。

注：「Proxy」策略组是用于访问国际互联网的默认策略，一般情况下，所有国际网络的访问都通过该策略组中选择的节点进行连接。

此时你就可以自由的穿梭于国际互联网之间啦！

Clash for Android 安装&教程

还是一样GitHub不稳定，下载链接：Android Clash for Android v2.4.9 中文版 | 六音软件 (sixyin.com)，选择对应的安卓版本，下载安装即可。

快速上手

Clash for Android 支持两种导入配置文档的方式：

  • URL （订阅）
  • 本地导入

首先复制好订阅链接，点解主页的配置按钮

然后点击右上角的加号按钮添加订阅链接

这里推荐选择从url导入，也就是黏贴进去复制的链接。

复制链接点击右上角保存按钮，即可配置成功。

当你返回去的时候，手机系统会问你要不要添加vpn只需点击允许就可以了。

返回主界面，点解开始按钮会看到代理选择，点进去

最后就可以选择你想要用的节点啦！
