Clash各大平台使用教程

Created2021-07-19|Updated2021-07-19
|Post View:

Clash简介

Clash是一款支持Windows、macOS、Android等多平台的代理软件，支持ss/v2ray/Trojan等，类Surge配置，macOS/Windows/Android上都已经有了图形界面，且召唤师已经支持ss/v2ray/Trojan的Clash订阅，使用起来已经非常简单，十分推荐使用。 简而言之就是一款代理软件，只要有订阅链接就能访问外网！接下来我将介绍如何在Windows，Mac上以及安卓手机上使用Clash。IOS系统暂不支持。

下载安装

  1. 源文件在Github, 但基于被墙，小伙伴们可以去Clash For Windows v0.16.3 客户端及汉化补丁 | 六音软件 (sixyin.com)下载 Windows 版本。选择对应的电脑版本就好了。Clash本身不是汉化版本，里面也有对应的汉化教程。

快速上手

  1. 使用方法也很简单，打开 Clash for Windows 应用程序，在左侧的标签页中选择 Profiles ， 在顶部输入您的 Clash 订阅链接，然后点击 Download 按钮。

Clash for Windows 会自动拉取配置文件进行更新，如果一切顺利，你应当可以看到成功的提示信息「Success!」，并且下方会自动多出一个文件。

点击新增的配置文件来切换到该配置，然后点击「Proxies」标签页来切换接入点，将顶部的出站模式选择为「Rule」。
此模式下你的网络访问请求将通过 Clash for Winows 进行分流处理。

返回到「General」部分，将「System Proxy」的开关更改为绿色或「On」状态即可开始使用。此外，如果您需要让 Clash for Windows 开机自启，可以将「Start with Windows」也更改为绿色或「On」。
